Default Ranges allow you to define the trend ranges that will be used as the defaults for Trends and Sparklines on Visual KPI trends. The available settings enable you to configure the default time ranges to be applied in all trends and charts available for end-users when accessing your Visual KPI instance. You find the Default Ranges Configuration under Site Settings (<PngFont name='options28' />) in the Document Actions panel.
